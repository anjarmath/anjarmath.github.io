Copy kode embed dari Google Form dan JotForm


  • Google Form
    1. Masuk ke halaman editing Google Form
    2. Klik tombol send di bagian atas
    3. pilih menu '<>', dan di situlah letak kode embed berada, silahkan copy

  • JotForm
    1. Masuk ke halaman editing JotForm
    2. Pilih menu Publish di bagian atas
    3. pilih menu 'Embed'
    4. ada beberapa pilihan, anda bisa mendapatkan kode embed dari bagian 'Embed' paling atas, namun karena kelengkapan source code disarankan memilih menu 'iframe'
    5. silahkan copy seluruh kode yang ada
